# 潮语拼音输入法

## 项目特色

- 提供六种口音版本：潮州、汕头、澄海、饶平、揭阳、潮阳
- 支持多种操作系统：Windows、macOS、Linux、Android、iOS
- 支持用汉语拼音（普通话）反查潮语拼音（潮汕话）
- 配套一份通俗易懂的《潮语拼音教程》
- 在 GitHub 上开源，欢迎有兴趣的爱好者参与
